# Animated Dashboard UI (Vanilla JS)

A simple front-end animation project built with **HTML, CSS, and Vanilla JavaScript**.  
The app animates UI elements sequentially on page load, creating a clean dashboard-style reveal.

On `window.load`, the script:
1. Reveals the header and app container
2. Rotates the spinner for a visual loading effect
3. Fades in the main content after a short delay
4. Animates multiple vertical bars by changing their heights dynamically

All animations are controlled using JavaScript with timed delays (`setTimeout`) and CSS transitions.
